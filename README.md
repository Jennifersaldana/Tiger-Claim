# Tiger-Claim

A React-based lost and found platform that lets LSU students securely report, upload, and search for lost items through LSUâ€™s SSO system.


## Setup
1. Download npm and node: https://nodejs.org/en/download

Type in Terminal:
```Bash
npm -v
node -v
```


## How to Run
1. Open the project in VScode ( Stay in the main branch)

``` Bash
git clone https://github.com/Jennifersaldana/Tiger-Claim.git
```

2. Open the web app

Type in Terminal:
```Bash
cd tigerclaim
npm install
npm start
```
