1. Download npm and node: https://nodejs.org/en/download



## How to Run
1. Open the project in VScode ( Stay in the main branch)

``` Bash
git clone https://github.com/Jennifersaldana/Tiger-Claim.git
```

2. Open the web app

Type in Terminal:
```Bash
npm start
```
